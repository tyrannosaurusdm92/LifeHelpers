OurSpace Shared DBT + ADHD Cadence

Start where the body and brain actually are. Tiny counts. Rest counts. Repair counts.

Daily

Start Day capacity check

  • Pick: recovery / tiny / medium / full.
  • Check breathing, pain, MCAS/sensory load, sleepiness, and caregiver capacity.
  • Choose one anchor only.

Done enough: The day is started when capacity is named, not when a task is finished.

One DBT/ADHD skill

  • Open one shared skill.
  • Do the 30-second version.
  • Mark partial completion.

Done enough: Reading the tiny version counts on flare days.

Us check-in

  • What do you need?
  • What capacity do I have?
  • What can be smaller?
  • What can wait?

Done enough: Both needs are acknowledged.

Reset after conflict/overload

  • Say: pause means I am coming back.
  • Set a return cue/time if possible.
  • Use one body-safe grounding option.

Done enough: No one has to chase reassurance through silence.

Weekly

Shared skill review

  • Pick one skill that helped.
  • Pick one that felt too hard.
  • Shrink the hard one.

Done enough: The system adapts to you.

Caregiver rest block

  • Put Jasper's rest on the plan.
  • Choose what will not be done during that rest.
  • Set up one backup/low-demand option.

Done enough: Care includes the caregiver.

Tiny environment reset

  • Pick one reachable zone.
  • Use MCAS-safe supplies.
  • Stop at five minutes or before symptoms rise.

Done enough: A safer tiny zone is enough.

Repair and reassurance check

  • Name one repair needed.
  • Name one appreciation.
  • Name one boundary for next week.

Done enough: Connection is maintained without overexplaining.

Monthly

Skill folder tune-up

  • Favorite the skills actually used.
  • Archive ones that shame or overwhelm.
  • Rewrite one prompt smaller.

Done enough: The shared folder stays alive and usable.

Supplies and access check

  • Check fragrance-free cleaning/hygiene supplies.
  • Check mobility/oxygen/reacher/shower/toileting setup.
  • Add one missing support to the list.

Done enough: Barriers are caught before crisis when possible.

Medical/admin gentle batch

  • Pick one form, appointment, refill, or phone call.
  • Body-double or split it.
  • Stop after one step.

Done enough: Admin is moved without consuming the whole month.

Yearly

Shared care language review

  • Confirm names/pronouns/endearments.
  • Confirm body-language boundaries.
  • Update scripts that no longer fit.

Done enough: The app respects who you are now.

Privacy and safety review

  • Check what the app stores.
  • Remove sensitive details that do not need to be visible.
  • Confirm links/pages do not expose private info.

Done enough: Support does not cost privacy.

Emergency/access documents review

  • Check emergency contacts and medical notes.
  • Check equipment/back-up needs.
  • Update only what changed.

Done enough: A future hard day has less chaos.
