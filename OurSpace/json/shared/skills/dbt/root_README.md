# OurSpace DBT + ADHD Shared Life-Filtered Pack

This pack revises the supplied DBT and ADHD skill files into a shared folder for William and Jasper/Squishy.

What changed:

- Added a shared life-filter to every JSON skill/worksheet/handout.
- Added a visible shared life-filter section to every fillable HTML file.
- Rewrote README introductions so the folders start from tiny-step, disability-first, trauma-informed use.
- Added `shared/life_filter/shared_life_filter.json`.
- Added `shared/routines/shared_dbt_adhd_cadence.json` and `.html` for daily, weekly, monthly, and yearly use.
- Shortened folder depth by flattening nested folders like `worksheets/html` into `worksheets_html`.

Core rules:

- Partial completion counts.
- Bed/chair-level options come first.
- MCAS/respiratory-safe and low-sensory defaults come first.
- Caregiver capacity is part of the task.
- Pause is not abandonment when repair is named.
- Skills are supports, not tests.

Private note: this pack uses the supplied life-impact profiles but does not copy the full profiles into the output.
