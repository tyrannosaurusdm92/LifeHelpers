# Licensing Notes

This package contains newly generated application glue code plus a small curated set of overlay image assets copied from `snapchat-filters-opencv-master`, which included an MIT license.

Copied license files are stored in `docs/licenses/`.

## Included third-party material

| Material | Source upload | License status |
| --- | --- | --- |
| Dog ears/nose/tongue, glasses, hat, mustache, rainbow, flies overlay sprites | `snapchat-filters-opencv-master.zip` | MIT license file present and copied to `docs/licenses/` |

## Reviewed but not copied into runnable code

MIT/BSD/GPL/license files from reviewed repositories are copied into `docs/licenses/` for audit trail where present. Copying a license file into documentation does not mean the corresponding source code was included in the runnable app.

## Excluded

Repositories that appeared to contain leaked/proprietary Snapchat source, account cracking, or unofficial account automation were not included in runnable code. If you distribute this package, review all licenses with your own legal counsel.
